About Waitless

Waitless is a web application designed by a team of developers as a final group project for a web development course. The application was created to streamline the process of ordering food at a Thai restaurant. It allows customers to place their orders ahead of time, avoid waiting in long lines, and make payments securely.

Overview of Project

The Waitless application was built using TypeScript, React, Node.js, and SQLite3. The app has a simple and intuitive user interface that guides customers through the ordering process. The front-end development team focused on creating a responsive design that looks great on both desktop and mobile devices. The back-end team worked on implementing the payment gateway and database integration.
